## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- A text editor(preferably Visual Studio Code)
- Node
- Web browser

### Using it Locally

- Clone the project

```bash 
git@github.com:vikipretium/motorcycle-testdrive-backend.git

cd motorcycle-testdrive

```

### Setup

Install gems with:

```
bundle install
```

Setup database with:

```
rails db:create
rails db:migrate
rails db:seed
```

### Usage

Start server with:

```
rails server
```

Visit http://localhost:3001/ in your browser.

### Run tests

Install npm with:

```
npm i
```

Install rspec with:

```
bundle install
```

and

```
rails generate rspec:install
```

run the test with:

```
rspec .
```
